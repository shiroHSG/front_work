속성 선택자

선택하고자 하는 요소내에 작성되어 있는 속성을 이용하여 선택하는 방법
선택자 뒤에 []를 이용하여 속성과 속성값을 제시함 

        선택자[속성명=속성값]{스타일 속성 : 값; ...} : 속성값이 "일치" 하는 요소
        선택자[속성명~=속성값]{스타일 속성 : 값; ...} : 속성값이 단어로 "포함"되어 있는 요소
        선택자[속성명^=속성값]{스타일 속성 : 값; ...} : 속성값으로 "시작"하는 요소
        선택자[속성명&=속성값]{스타일 속성 : 값; ...} : 속성값으로 "끝"나는 요소
        선택자[속성명*=속성값]{스타일 속성 : 값; ...} : 속성값이 "포함"하는 요소
